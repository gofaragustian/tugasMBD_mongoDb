| Operator | Fungsi                                     |
| -------- | ------------------------------------------ |
| `$and`   | Semua kondisi harus terpenuhi              |
| `$or`    | Salah satu kondisi cukup terpenuhi         |
| `$not`   | Membalikkan hasil dari kondisi             |
| `$nor`   | Tidak satupun dari kondisi boleh terpenuhi |
